# Security Policy

## Підтримувані версії

| Версія | Підтримка безпеки |
|--------|------------------|
| 1.3.x  | ✅ Активна       |
| < 1.3  | ❌ Не підтримується |

## Повідомлення про вразливості

**НЕ** публікуйте вразливості безпеки як публічні GitHub Issues.

Надішліть звіт на: **root_bsd@itprof.net.ua**

Вкажіть:
- Опис вразливості
- Кроки для відтворення
- Потенційний вплив
- Версію ScanXSS

Відповідь надійде протягом 48 годин.

## Відмова від відповідальності

ScanXSS призначений **виключно** для авторизованого тестування безпеки.
Використання проти систем без явного дозволу власника є незаконним.

Автори не несуть відповідальності за будь-яке неправомірне використання.
