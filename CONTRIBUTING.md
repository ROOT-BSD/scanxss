# Contributing to ScanXSS

Дякуємо за інтерес до проєкту!

## Як зробити внесок

### Bug Reports
Відкрийте Issue з:
- Версією ScanXSS (`./scanxss --version`)
- ОС і версією
- Кроками для відтворення
- Очікуваним і фактичним результатом

### Pull Requests

1. **Fork** репозиторію
2. Створіть гілку: `git checkout -b fix/your-bug` або `feat/your-feature`
3. Зробіть зміни, переконайтесь що збірка чиста:
   ```bash
   cd linux && make clean && make
   ```
4. Відкрийте PR з описом змін

### Стиль коду

- Стандарт: C99
- Відступи: 4 пробіли
- Максимальна довжина рядка: 100 символів
- Нові модулі атак — у `linux/modules/mod_*.c`
- Нові утиліти — у `linux/src/`

### Нові модулі атак

Кожен модуль реалізує функцію:
```c
int module_NAME_run(ScanContext *ctx, const Form *form);
```

Payload-и додаються через `payloads_add()` у `payloads.db` —
не хардкодьте масиви безпосередньо у код.

### Відповідальне використання

Всі контрибутори погоджуються використовувати і розробляти
ScanXSS виключно для **авторизованого** тестування безпеки.
